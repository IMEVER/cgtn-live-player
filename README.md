# cgtn-live-player
A qt client for cgtn live player.
* [cgtn website](http://www.cgtn.com)
* [cgtn stream](https://live.cgtn.com/manifest.m3u8)

This app depends some deepin's dtk package and some qt5 package
- libdtkbase
- libdtkwidget
- libdtkutil,
- libqt5core5a
- libqt5gui5
- libqt5widgets5
- libqt5dxcb-plugin
- libqt5multimedia5
